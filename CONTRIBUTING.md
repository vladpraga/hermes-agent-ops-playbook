# Contributing

Contributions are welcome, especially practical improvements to local-first AI agent operations.

Useful contributions include:

- clearer diagnostic checklists;
- safer shell/Python examples;
- no-agent watchdog patterns;
- gateway delivery troubleshooting notes;
- memory hygiene examples;
- documentation fixes.

Please avoid adding secrets, private logs, private chat IDs, or machine-specific credentials to issues or pull requests.
